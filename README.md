# Метод Шварца: сравнение аддитивной и мультипликативной схем

Курсовая работа по направлению 01.03.04 «Прикладная математика» (КФУ,
ИВМиИТ). В работе реализованы и сопоставлены два варианта итерационного
метода декомпозиции области (метода Шварца) применительно к стационарной
краевой задаче в одно- и двумерной постановке.

## Структура каталога

```

├── scripts/     головные циклы метода Шварца

│   ├── schwartz_add_step.m

```

## Запуск

Требуется MATLAB R2022a или новее (используются возможности `uifigure` и
параллельной обработки `parfor`).

Одномерное приложение:

```matlab
cd apps
schwartzapp
```

Двумерное приложение:

```matlab
cd apps
schwartzapp2d
```

В графическом интерфейсе можно настроить число узлов, точность,
радиус пересечения, формулу правой части, краевые условия и переключаться
между мультипликативной и аддитивной схемами.

## Сборка пояснительной записки

В каталоге `report/` достаточно стандартного дистрибутива TeX Live или
MiKTeX с пакетами `babel`, `biblatex`, `biber`, `tikz`, `microtype` и
`hyperref`:

```bash
cd report
pdflatex report.tex
biber report
pdflatex report.tex
pdflatex report.tex
```

## Краткое содержание работы

* Постановка одномерной и двумерной краевых задач для уравнения энергии,
  разностная аппроксимация.
* Описание мультипликативной и аддитивной итерационных схем метода Шварца.
* Эксперименты с радиусом пересечения подобластей, числом подобластей,
  распределением узлов сетки, использованием метода Зейделя в качестве
  локального решателя.
* Сравнение схем по числу итераций и реальному времени работы для
  одномерной и двумерной задач.

Подробнее --- в `report/report.pdf`.
